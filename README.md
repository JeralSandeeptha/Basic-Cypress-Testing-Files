## Prerequiesites

Cypress requires `Nodejs` Run Time Environment. First we need to install `Nodejs`.

## Install Cypress

```bash
npm i -D cypress
```

## Run Cypress

Through Cypress Application
```bash
npx open cypress
```

Thorugh Terminal
```bash
npx cypress run
```
